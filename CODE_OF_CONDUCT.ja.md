# 行動規範（Contributor Covenant）

## 私たちの誓い

メンバー、コントリビューター、リーダーとして、私たちはコミュニティへの参加をすべての人にとって歓迎できる体験にすることを誓います。

## 私たちの基準

コミュニティにとって良い環境を作るための行動の例として、以下が挙げられます。

* 他者に対して共感と親切心を示す
* 異なる意見・見方・経験を尊重する
* 建設的なフィードバックを丁寧に与え、また丁寧に受け入れる
* 自分のミスによって影響を受けた人に責任を認めて謝罪し、そこから学ぶ
* 個人としてだけでなく、コミュニティ全体にとって最善のことに注力する

許容できない行動の例として、以下が挙げられます。

* 不適切な言葉や画像の使用
* トローリングや侮辱的・軽蔑的なコメント
* 公的または私的な不適切な行為
* 明示的な許可なく他者の個人情報を公開すること
* 職業的な場において不適切とみなされる可能性のある、その他の行為

## 施行の責任

コミュニティリーダーは、受け入れ可能な行動の基準を明確にし、施行する責任を持ちます。不適切・脅迫的・攻撃的・有害と判断した行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

本行動規範は、すべてのコミュニティスペースで適用されます。また、個人がコミュニティを公式に代表して公的スペースで活動する場合にも適用されます。

## 施行

虐待的・嫌がらせ・その他許容できない行動が認められた場合は、施行担当のコミュニティリーダーに [GitHub Issues](https://github.com/tackeyy/xai-cli/issues) から報告してください。

すべての申告は迅速かつ公正に審査・調査されます。

コミュニティリーダーは、報告者のプライバシーとセキュリティを尊重する義務を負います。
